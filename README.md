# NameFinder
عطها لستة ارقام وتعطيك اساميهم

# install on linux 
- Clone this repository
```
$ git clone https://github.com/KalidOp/NameFinder.git 
```
- Run!
```
$ cd NameFinder
```
```
$ python3 NameFinder.py
```
